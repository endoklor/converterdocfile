DOC/DOCX to PDF Converter
Описание

Программа на Python для конвертации файлов форматов .doc и .docx в PDF. Поддерживает как выбор отдельных файлов, так и пакетную обработку папок с сохранением структуры каталогов. Оснащена графическим интерфейсом для удобного использования.

Возможности
Конвертация файлов .doc и .docx в PDF
Выбор отдельных файлов через интерфейс
Обработка целых папок
Сохранение структуры вложенных папок
Автоматическое создание папки с PDF-файлами
Отображение процесса выполнения
Логирование ошибок
Требования
Python 3.8 и выше
Windows (рекомендуется, если используется Microsoft Word)
Установленный Microsoft Word (если используется конвертация через COM)
Установка и запуск
Быстрый запуск

python run.py

Ручной запуск
Установить зависимости:
python install.py
Запустить программу:
python main.py
Структура проекта

project/
│
├── main.py # Основной файл приложения (GUI и логика)
├── install.py # Скрипт установки зависимостей

Возможные ошибки и решения

Ошибка: Microsoft Word не найден
Причина: библиотека требует установленный Word
Решение: установить Microsoft Word или использовать альтернативную библиотеку

Ошибка: не устанавливаются зависимости
Причина: отсутствует интернет или pip
Решение:
python -m ensurepip --upgrade
python -m pip install --upgrade pip

Ошибка доступа к файлам
Причина: нет прав или файл открыт
Решение: закрыть файл и проверить права доступа
